How to compile/run examples
------------------------------------
    cd examples
    mkdir build
    cd build
    cmake ..
    make
    ./dambreak

How to compile/run tests
--------------------------
    cd tests
    mkdir build
    cd build
    cmake ..
    make test

How to compile the library
----------------------------
    cd hokusai
    mkdir build
    cmake ..
    make

How to generate the documentation
-----------------------------------
    cd hokusai
    mkdir build
    cmake ..
    make doc
